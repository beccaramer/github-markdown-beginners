# Getting Started with GitHub & Markdown for Beginners

**Author:** Becca Ramer

## Summary
This tutorial introduces beginners to GitHub and Markdown, two essential tools used in technology and software-related fields. It explains what GitHub is, why it is important, and how Markdown is used to create clear and professional documentation. By the end of the tutorial, users will understand how to create a GitHub repository and write a basic README file using Markdown.

## Target Audience
This tutorial is intended for beginners with little to no prior experience using GitHub or Markdown. It is especially helpful for undergraduate students, non-technical majors, or anyone new to version control and technical documentation.

## What You Will Learn
- What GitHub is and how it is used
- What Markdown is and why it matters
- How to create a basic GitHub repository
- How to write and format a README.md file
