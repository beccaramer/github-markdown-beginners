# What Is GitHub?

[← Back to Home](README.md) | [Next → Markdown Basics](markdown-basics.md)

## Overview
GitHub is a web-based platform used to store, manage, and share code and digital projects. It uses a system called Git, which tracks changes made to files over time.

## Why GitHub Is Important
- Stores project files in one place
- Tracks changes and revisions
- Allows collaboration with others
- Commonly used by employers and teams

## Key Terms
- **Repository (Repo):** A project folder stored on GitHub
- **Commit:** A saved change to a file
- **README:** A file that explains what a project is and how to use it
