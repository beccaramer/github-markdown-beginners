# Creating Your First GitHub Repository

[← Back](markdown-basics.md) | [Next → Next Steps](next-steps.md)

## Step 1: Create a GitHub Account
Go to https://github.com and create a free account if you do not already have one.

## Step 2: Create a New Repository
1. Click the **New** button on your GitHub dashboard
2. Enter a repository name
3. Select **Public**
4. Check the box to add a README file
5. Click **Create Repository**

## Step 3: Edit the README
- Click on `README.md`
- Select the pencil icon to edit
- Use Markdown to format text
- Save changes by committing them
